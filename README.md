# fuzzy_bvp_labeler

Download folder with your name and add it to repo to `data` directory. 

Next install requirements and run `streamlit run streamlit_labeller.py`.

For egzample:
```
fuzzy_bvp_labeler
│   README.md
│   streamlit_labeller.py
└───src 
│   │   config.py
│   │   create_dataset.py
│
└───data # contains all the data
│   └─── Karo
│   │   └─── samples
```
